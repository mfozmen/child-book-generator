# docs/

Internal planning notes. Each task is a single Markdown file. **When a task ships, delete the file.** The set of files in this directory is the set of open work.

Naming: `p<phase>-<nn>-<slug>.md`. Phase 1 = PDF-first ingestion. Phase 2 = layout improvements (not started until Phase 1 is done).

Nothing in this folder is user-facing. User docs live in the root `README.md`.
